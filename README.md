
# Salsamentería Almendra

## Descripción general

Proyecto de página web informativa y de catálogo para una salsamentería local llamada "Salsamentería Almendra". Presenta el portafolio de productos (carnes, huevos, quesos y otros), información de contacto y un formulario para mensajes. Está orientado a clientes locales que deseen conocer la oferta de productos y contactar al negocio.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript (módulos ES6)

## Estructura del proyecto

- `index.html`: Página principal con sección de inicio, productos destacados y formulario de contacto.
- `productos/index.html`: Página de catálogo con filtros por categoría y listado completo de productos.
- `css/styles.css`: Estilos globales y específicos (diseño responsivo, modal, toasts, botón de WhatsApp).
- `js/script.js`: Lógica de la interfaz (navegación, formulario, notificaciones tipo toast, modal de beneficios, interacciones y eventos).
- `js/data.js`: Datos de ejemplo (objeto `beneficiosProductos`) con beneficios descriptivos por producto.
- `images/`: Imágenes de los productos utilizadas en ambas páginas.
```

## Funcionalidades principales

- Catalogo de productos con imágenes, descripción y precios.
- Página de productos con filtros por categoría (`Todos`, `Carnes`, `Huevos`, `Otros`).
- Modal informativo que muestra los beneficios de cada producto (datos en `js/data.js`).
- Formulario de contacto con notificación tipo "toast" al enviar.
- Botón flotante de WhatsApp para contacto rápido.
- Diseño responsivo y accesible para visualización en dispositivos móviles y de escritorio.

## Estado del proyecto

En desarrollo (demo funcional): la interfaz y las funciones principales están implementadas en el cliente. Puede ampliarse con un backend para procesar mensajes del formulario, administración de productos y un sistema de pedidos.
